# Dot Files
### Overview
Various configuration 'dot' files, such as `.bashrc`, `.vimrc`, etc.

### Making Links
`ln -s ~/dot_files/.bashrc .bashrc`<br>
`ln -s ~/dot_files/.vimrc .vimrc`

### Miscellaneous
 * To change the color of comments in vim `:hi Comment ctermfg=<color>`
